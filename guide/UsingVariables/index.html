Personalize WhatsApp Messages with Variables

Learn how to use variables to send personalized bulk messages to each contact

What Are Message Variables?

Variables are placeholders you add to your message template using curly braces {columnName}. When the message is sent, WasappBulk automatically replaces each placeholder with that contact's actual data from your Excel file.

This lets you send personalized WhatsApp messages in bulk — greeting each person by name, including their order details, scores, city, or any custom field, all from a single message template without writing each message manually.

Simple Example

❌ Instead of writing:

• "Hi John, you scored 95!"

• "Hi Sarah, you scored 87!"

• "Hi Mike, you scored 92!"

✅ You write ONE template:
"Hi {name}, you scored {marks}!"

WasappBulk automatically sends personalized versions to each person

How It Works

  1. Prepare Excel file with column headers
    Create an Excel file where the first row is headers. Example: Name, Phone, Marks, City
  2. Upload Excel to WasappBulk
    Click 'Upload Excel' — the extension reads all column headers and imports your contacts with their data.
  3. Write message with {placeholders}
    Type your message and use {columnName} placeholders wherever you want personalized data.
  4. Click Send
    WasappBulk automatically replaces each placeholder with that person's actual data and sends the personalized message.

Any Excel Column Works!

You're not limited to "Name" and "Marks". Any column header in your Excel becomes a variable. Here are common examples:

Column Header Use in Message Example
Name {name} John, Sarah, Mike
Email {email} john@test.com
Marks {marks} 95, 87, 92
City {city} Mumbai, Delhi, Pune
Company {company} TechCorp, DesignHub
OrderID {orderid} ORD12345
Amount {amount} 5000, 2500
Package {package} Premium, Standard

Common Mistakes

❌ Wrong spelling:

Message: "Hi {nmae}, congratulations!"

Result: "Hi {nmae}, congratulations!" — NOT replaced (typo!)

✅ Correct spelling:

Message: "Hi {name}, congratulations!"

Result: "Hi John, congratulations!"

Syntax Rules

  • Use curly braces: {name} ✓ | (name) ❌ | [name] ❌
  • Match column names: If Excel has 'Name', use {name}
  • No spaces inside: {name} ✓ | { name } ❌
  • Case-insensitive: {Name}, {name}, {NAME} all work
  • Use any column: {phone}, {marks}, {city}, {company}

Tips & Best Practices

  • You can use multiple variables in one message — as many as you want
  • If a contact has no value for a variable, it's replaced with empty text — no broken messages
  • Works with .xlsx, .xls, .csv files — any spreadsheet format
  • Your Excel's first row MUST be column headers for variables to be detected
  • Double-check spelling — variables must match your Excel column headers exactly
